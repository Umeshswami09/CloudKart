# CloudKart

## Enterprise E-Commerce Platform on AWS

### 📌 Project Description

CloudKart is a production-ready Enterprise E-Commerce Website developed using Django and deployed on Amazon Web Services (AWS).

This project demonstrates a secure and scalable three-tier architecture using AWS cloud services.

---

## 🚀 Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript
- Bootstrap

### Backend
- Django

### Database
- PostgreSQL (Amazon RDS)

### Cloud Platform
- Amazon Web Services (AWS)

---

- Amazon EC2

- Amazon S3

- Auto Scaling

## 🎯 Project Objective

To build and deploy a secure, scalable and highly available e-commerce application on AWS using industry-standard cloud architecture.
