About CloudKart

CloudKart is an Enterprise E-Commerce Platform developed using Django Framework and AWS Cloud Services. The project demonstrates secure authentication, shopping cart management, checkout system, order management and cloud deployment using modern AWS architecture.

🎯 Our Mission

To build a secure, scalable and cloud-based e-commerce platform using modern AWS services and Django.

🌍 Our Vision

To provide users with a fast, secure and reliable online shopping experience powered by cloud computing.
