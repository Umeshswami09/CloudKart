# Session 01

## Topic
Static Files (CSS)

## Purpose

- CSS website ki styling ke liye use hoti hai.
- Django me CSS ko `static` folder ke andar rakha jata hai.
- `STATIC_URL` aur `STATICFILES_DIRS` ki help se Django CSS, JavaScript aur Images serve karta hai.
- Is step ke baad website plain HTML se professional UI me convert hone lagti hai.
